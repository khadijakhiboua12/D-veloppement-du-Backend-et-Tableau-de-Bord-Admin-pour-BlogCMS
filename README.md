# BlogCMS - Backend PHP & MySQL

## Description
BlogCMS est une application de gestion de contenu pour blogs développée en **PHP procédural** avec **MySQL**.  
Elle permet aux administrateurs, auteurs et visiteurs de gérer et consulter les articles, catégories, utilisateurs et commentaires.

---

## Fonctionnalités

### Pour tous les utilisateurs
- Page de login sécurisée
- Système de rôles : `admin`, `éditeur`, `utilisateur`

### Administrateurs
- Dashboard avec statistiques
- Gestion des catégories (CRUD)
- Gestion des utilisateurs (CRUD)
- Modération des commentaires

### Auteurs
- Voir, créer, éditer et supprimer ses articles
- Poster des commentaires

### Visiteurs
- Voir les articles publiés
- Poster des commentaires (modérés par l’admin)

---

## Technologies utilisées
- **PHP procédural** pour le backend
- **MySQL** pour la base de données
- **HTML / CSS / Bootstrap** pour le frontend
- **PDO** pour les requêtes sécurisées
- **Sessions PHP** pour la gestion de l’authentification
